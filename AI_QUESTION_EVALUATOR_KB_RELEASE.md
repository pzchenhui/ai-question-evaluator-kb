## ✨ 核心升级

### 1. 知识库系统（6 大类）

| 知识库类型 | 文件 | 内容 | 作用 |
|------------|------|------|------|
| **课程标准** | curriculum_standards.json | 小学/初中/高中各学科课程标准 | 适切性评价依据 |
| **知识点体系** | knowledge_points.json | 知识点名称、重要程度、前置知识 | 知识点覆盖分析 |
| **常见错误库** | common_errors.json | 概念错误、计算错误、审题错误 | 科学性评价增强 |
| **典型例题** | example_questions.json | 各学科典型例题（含难度标注） | 难度参考对比 |
| **认知要求** | cognitive_requirements.json | 各学段认知层次上限要求 | 认知层次评价依据 |
| **难度参考** | difficulty_reference.json | 各学段难度区间参考 | 难度预估校准 |

### 2. 评价增强功能

| 功能 | v1.0 | v1.1.0 | 提升 |
|------|------|--------|------|
| 科学性评价 | 基础规则匹配 | + 常见错误库匹配 | +15% |
| 适切性评价 | 题干长度判断 | + 课程标准对照 | +20% |
| 认知层次 | 关键词匹配 | + 学段认知要求 | +15% |
| 知识点分析 | ❌ | ✅ 覆盖分析 | 新增 |
| 错误预警 | ❌ | ✅ 常见错误预警 | 新增 |
| 难度预估 | 简单估算 | + 知识库校准 | +10% |

### 3. 知识库可扩展

支持用户自定义添加知识：

```python
from ai_question_evaluator_kb import KnowledgeBase

kb = KnowledgeBase()

# 添加新的常见错误
kb.add_knowledge("common_errors", {
    "物理": {
        "概念错误": [
            {
                "error": "混淆速度和加速度",
                "example": "认为速度大加速度就大",
                "correction": "速度描述运动快慢，加速度描述速度变化快慢"
            }
        ]
    }
})

# 添加新的知识点
kb.add_knowledge("knowledge_points", {
    "地理": {
        "初中一年级": [
            {
                "name": "等高线地形图",
                "level": "核心",
                "prerequisites": ["地图比例尺", "海拔概念"]
            }
        ]
    }
})
```

## 🚀 使用方式

### 方式一：SkillHub 安装

```bash
skillhub install ai-question-evaluator-kb
```

### 方式二：命令行直接使用

```bash
# 单题评价
python3 ai_question_evaluator_kb.py \
  -q "题目内容" \
  -a "答案" \
  -s 数学 \
  -g 初中二年级 \
  -t 选择题

# 批量评价
python3 ai_question_evaluator_kb.py -f questions.json

# 禁用知识库（基础版）
python3 ai_question_evaluator_kb.py -q "题目" --no-kb
```

### 方式三：Python API

```python
from ai_question_evaluator_kb import QuestionEvaluatorKB

evaluator = QuestionEvaluatorKB({
    "subject": "数学",
    "grade": "初中二年级",
    "enable_kb": True
})

report = evaluator.evaluate(
    question="已知一次函数 y=kx+b 的图象经过点 (1,3) 和 (2,5)，求解析式。",
    answer="y=2x+1"
)

print(f"综合得分：{report['综合得分']}分")
```

## 🛠️ 技术实现

### 核心类

1. **QuestionEvaluatorKB**: 题目评价器（继承 v1.0）
2. **KnowledgeBase**: 知识库管理器（新增）

### 关键算法

1. **知识点匹配**: 基于关键词的知识点识别
2. **错误预警**: 题目文本与常见错误库匹配
3. **难度校准**: 基于知识库的难度预估修正

### 依赖库

- `jieba`: 中文分词（可选，用于更精准的知识点匹配）
- 标准库：`json`, `re`, `datetime`, `pathlib`

## 🎓 教育理论基础

### 1. 教育测量理论

- **效度**: 题目是否测量了目标知识/能力
- **信度**: 题目测量结果的稳定性
- **难度**: 题目的难易程度（P 值）
- **区分度**: 题目区分不同水平学生的能力

### 2. 布鲁姆教育目标分类

| 层次 | 认知过程 | 典型动词 |
|------|----------|----------|
| 记忆 | 回忆知识 | 说出、写出、列举 |
| 理解 | 理解意义 | 解释、说明、概括 |
| 应用 | 应用知识 | 运用、计算、解决 |
| 分析 | 分析结构 | 分析、区分、推断 |
| 评价 | 价值判断 | 评价、评判、论证 |
| 创造 | 创造新知 | 设计、创造、构建 |

### 3. 课程标准依据

- 义务教育课程标准（2022 年版）
- 普通高中课程标准（2017 年版 2020 年修订）
- 各学科教学指导意见
